## Gerenciador de Tarefas - Versão 1

Este foi meu primeiro projeto utilizando Spring Boot integrado com Thymeleaf para renderização de páginas web.

O objetivo principal deste projeto foi praticar os conceitos fundamentais do desenvolvimento backend com Java, aplicando o padrão de arquitetura MVC (Model-View-Controller) e entendendo o fluxo completo entre Controller, Service, Repository e View.

## Funcionalidades

- Cadastro de tarefas
- Listagem de tarefas
- Edição de tarefas
- Exclusão de tarefas

## Arquitetura

### O projeto segue o padrão MVC:

- Model → representa a estrutura da tarefa
- Controller → recebe as requisições HTTP e controla o fluxo da aplicação
- View (Thymeleaf) → renderiza as páginas HTML dinamicamente

## Tecnologias Utilizadas

- Java
- Spring Boot
- Spring MVC
- Thymeleaf
- Bootstrap
